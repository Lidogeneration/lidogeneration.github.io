Новая почта

- Возможность встраивания в любую конфигурацию

- Функции

- Загрузка всех справочников новой почты, обновление данных

- Автоматическое создание отправителей новой почты при создании клиентов в базе

- Создание ТТН Новой почты, отслеживание всех этапов отправлений от создания до получения отправления клиентами, в случае наложенного платежа, получения обратной доставки денежных средств от клиента

- Печать ТТН, печать маркировки

- Создание и ведение реестра отправок

- Автоматизация отправки писем на любом из этапов отправления

- Разделение заказов по этапам отправления: Новый, В работе, Оплата, Ожидает отправки, Ожидание обратных доставок, Учет отправленных посылок: Наложкой, Не наложкой, Отказы клиентов

- Учет не забранных клиентов

- Получение суммы доставки

- Расчет примерного дня доставки

- Создание заявлений на возврат отправлений

Автоматическое изменение статусов заказов для сайтов на Prom.ua

Обработка позволяет изменять статус заказа в личном кабинете Prom.ua при наступлении какого либо события
Например:

При проведении заказа статус может быть сменен с Новый на Принятый

При смене статуса на Отменен в кабинете он также становится отмененным

При установке статуса Отгружен в кабинете автоматически устанавливается статус - Выполнен

Так же есть возможность настроить смену статуса по тем условиям которые нужны именно Вам
